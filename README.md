# Password Generator

A simple web-based password generator built using HTML, CSS, and JavaScript.
This project demonstrates basic JavaScript logic, DOM manipulation, and user interaction.

## Features
- Generates random secure passwords
- User-friendly interface
- Instant password generation

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project was built to strengthen my understanding of JavaScript logic and
front-end development as part of my learning journey in computer science.
